Git

  • git ls-files - используется в Git для отображения списка файлов, которые отслеживаются репозиторием
  • git switch - команда для переключения между ветками
  • git status - команда в Git, которая отображает текущее состояние локального репозитория
  • git init - это команда для инициализации нового репозитория, которая создает скрытую папку .git в текущей директории проекта
  • git mv - перемещение или переименование файлов
